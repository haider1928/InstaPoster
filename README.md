# 📸 Instagram Hadith & News Auto Poster

This project fetches daily Hadith and trending global News headlines, formats them, and posts directly to your Instagram account. It runs automatically on a schedule using GitHub Actions and is perfect for spiritually or socially driven content accounts.

---

## ⚙️ Features

- 🕋 Daily Hadith auto-fetch and post
- 🌐 Top daily news headlines via [NewsAPI](https://newsapi.org/)
- 🕰️ Scheduled posting with GitHub Actions + Cron
- 🔒 Secure credential management using `.env`
- 🖼️ Instagram auto-post integration
